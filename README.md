# parallel

並列分散処理、最終課題用のソースコードと実行用ファイルをまとめている。
 
# Usage

git内にある実行ファイルを直接実行する。もしくは、
go言語の環境を整えた上でgoファイルをrunすることで実行できる。
 
git clone https://github.com/hoge/~
cd n_queen
go run n_queen.go
------ or -----
go run para_n_queen.go
